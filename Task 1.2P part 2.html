JavaScript Methods Demo

String Methods

JavaScript provides several string methods to manipulate text. Some of them include:

  • length: Returns the length of the string.
  • toUpperCase(): Converts the string to uppercase.
  • toLowerCase(): Converts the string to lowercase.
  • substring(startIndex, endIndex): Extracts a portion of the string.
  • indexOf(searchValue): Returns the index of the first occurrence of a substring.

Number Methods

JavaScript provides several number methods for mathematical operations and formatting. Some of them include:

  • toFixed(digits): Formats a number with a fixed number of decimal places.
  • toPrecision(digits): returns a string, with a number written with a specified length:.
  • toExponential(digits): Converts a number to exponential notation.
  • Math.max(...numbers): Returns the maximum value from a list of numbers.
  • Math.min(...numbers): Returns the minimum value from a list of numbers.

Array Methods

JavaScript provides several array methods to manipulate and work with arrays. Some of them include:

  • length: Returns the number of elements in the array.
  • join(separator): Joins all elements of an array into a string with the specified separator.
  • push(element1, ..., elementN): Adds one or more elements to the end of an array and returns the new length.
  • pop(): Removes the last element from an array and returns that element.
  • indexOf(searchElement, fromIndex): Returns the first index at which a given element can be found in the array.

Date Methods

JavaScript provides several date methods to work with dates and times. Some of them include:

  • toLocaleDateString(): Returns the date portion of a Date object as a human-readable string in the local time zone.
  • toLocaleTimeString(): Returns the time portion of a Date object as a human-readable string in the local time zone.
  • getDay(): Returns the day of the week (0 for Sunday, 1 for Monday, etc.).
  • getMonth(): Returns the month (0 for January, 1 for February, etc.).
  • getFullYear(): Returns the year of the date as a four-digit number.

Function Methods

JavaScript provides several function methods for handling functions. Some of them include:

  • call(thisArg, arg1, arg2, ...): Calls a function with a given this value and arguments provided individually.
  • apply(thisArg, [argsArray]): Calls a function with a given this value and an array of arguments.
